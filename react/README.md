# React Frontend

Frontend này là dashboard demo cho hệ thống mining.

Chức năng scaffold:

- xem summary mô hình từ Java backend
- bấm train lại mô hình
- nhập thông tin một khách hàng và gọi dự đoán churn

API đang dùng:

- `GET /api/dashboard/summary`
- `POST /api/models/train`
- `POST /api/predictions`

Lệnh chạy sau khi cài npm:

```bash
npm install
npm run dev
```
